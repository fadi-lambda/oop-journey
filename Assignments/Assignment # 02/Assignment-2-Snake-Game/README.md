# 2D Snake Game 🐍

A 2-player Snake game built with SFML in C++

## Features
- 2 player mode
- Leaderboard system
- Smooth graphics with SFML

## Controls
**Player 1:** W A S D
**Player 2:** Arrow Keys

## Requirements
- SFML library installed

## How to Run
1. Install SFML
2. Compile all .cpp files
3. Run the executable
