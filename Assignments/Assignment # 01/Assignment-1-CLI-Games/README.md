# CLI Games Hub 🎮

A collection of 5 CLI games built in C++ with colorful interface using Windows.h

## Games
1. 🎯 TicTacToe
2. 🐍 Snake & Ladder
3. 🃏 Match Cards
4. 🔢 Number Guessing Game
5. ✂️ Rock Paper Scissors

## Features
- Full leaderboard system
- Leaderboard by individual game
- Colorful CLI interface using Windows.h
- Player vs Player & Player vs Computer

## How to Run
- Compile all .cpp files
- Run main.cpp
- Windows only (uses Windows.h for colors)
